# SC2 Co-op Community Mirror

This is a community mirror of [starcraft2coop.com](https://starcraft2coop.com), which shut down at the end of March 2026.

The site serves as a repository of Starcraft II co-op information including commander guides, mission data, resources, and more — all gathered by the SC2 co-op community over the years.
